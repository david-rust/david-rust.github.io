Technical Support & Access

Secure Submission
Online Secure File Transfer

The KCR File Exchange System (FES) is the required method for submitting cancer case data files to the registry. All data transmissions are encrypted and authenticated through the UK FES portal.

Open FES Portal
fes.uky.edu — UK secure file exchange

Registry Software
CPDMS.net

CPDMS.net is KCR's web-based cancer data management system used by reporting facilities to enter, manage, and submit cancer abstracts. Access requires a registered account issued by KCR.

Public Data Tool
cancer-rates.info Support

cancer-rates.info is KCR's public-facing interactive cancer statistics tool, providing county-level cancer incidence and mortality rates for Kentucky. Support resources include documentation, the iOS mobile app, and direct contact with KCR staff.
