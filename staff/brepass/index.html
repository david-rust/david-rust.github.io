Biography

About Branson

Branson Repass serves as Senior Systems Programmer for the Kentucky Cancer Registry, leading programming efforts for core registry systems and data processing pipelines.

He develops, tests, and maintains complex software components that support cancer case processing, data validation, and interoperability with state and national cancer surveillance networks.
